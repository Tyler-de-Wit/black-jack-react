# Black Jack

Black Jack application built with React.

## Description

Single Page Application where a user can play black jack against the computer.

## Getting Started

### Dependencies

- npm

### Installing

- Clone the repository

### Executing program

- Install the dependencies
- Start the dev server

```bash
npm install
npm run dev
```
